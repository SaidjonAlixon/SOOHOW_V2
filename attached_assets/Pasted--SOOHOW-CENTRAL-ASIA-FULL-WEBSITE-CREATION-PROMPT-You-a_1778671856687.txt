## 3D EFFECTS — THREE.JS

### Hero Background — Molecular Network

In the hero section, render a full-screen Three.js canvas behind all content. Create a dynamic 3D molecular/network visualization:

- 150–200 nodes (small glowing spheres, Electric Blue `#00A8E8`, radius 0.04)
- Nodes connected by thin lines when within distance threshold (line color: `rgba(0,168,232,0.15)`)
- Entire structure slowly rotates on Y and X axes
- On mouse move: the entire network subtly tilts toward the cursor (parallax-like)
- On mouse click: a ripple pulse expands from click point, pushing nearby nodes outward then back
- Camera: PerspectiveCamera, z=8, no orbit controls — auto-animation only
- Renderer: `alpha: true`, `antialias: true`, transparent background
- Resize handler: canvas always fills viewport

This creates a living, breathing molecular world — perfect for a chemical reagents and instruments company.

### Product Cards — Vanilla Tilt 3D

Every product card gets `data-tilt` with:
- max tilt: 8 degrees
- scale: 1.03 on hover
- glare effect: enabled, max-glare 0.15, white glare
- speed: 400ms
- Perspective: 1000px

The card must feel like a physical object you can pick up and turn.

### About Section — Floating 3D Orbs

In the About section background, render 3 large semi-transparent glowing spheres using Three.js or pure CSS 3D transforms. They slowly drift and pulse. This adds depth without distracting from content.

## SCROLL ANIMATIONS — GSAP + SCROLLTRIGGER

Register `ScrollTrigger` plugin. Apply the following:

### Section Titles
Every `h2` section title animates on scroll:
- Characters split into individual spans
- Each character drops from `y: -60` with `opacity: 0`
- Stagger: `0.03s` per character
- Duration: `0.6s`, ease: `power3.out`

### Product Cards
- Cards enter from bottom: `y: 80, opacity: 0`
- Stagger: `0.12s` between cards
- Trigger: when card's top hits 85% of viewport
- Ease: `power2.out`

### Statistics Counter
Numbers count up from 0 when scrolled into view using `CountUp.js` or custom interval function. Each number has a suffix ("+", "K+", "%"). Duration: 2 seconds per counter.

### Industrial Timeline (About section)
- Vertical line draws downward using `scaleY` from 0 to 1, `transformOrigin: top`
- Each milestone fades in from left or right alternately as the line reaches it
- Smooth, sequential reveal

### Partner Logos Carousel
- Infinite horizontal scroll using CSS animation `@keyframes scrollLeft`
- Two rows: top row scrolls left, bottom row scrolls right
- Speed: 35s per loop
- On hover: pauses (`animation-play-state: paused`)
- Images grayscale by default, full color on hover

### CTA Section Parallax
- Background moves at 60% scroll speed using GSAP ScrollTrigger scrub
- Text moves at 100% (normal)
- Creates depth illusion

### Number Badges
When spec numbers scroll into view, they count up:
```
500+  →  Industrial Instruments
200+  →  Satisfied Clients
15+   →  Partner Brands
10+   →  Years of Experience
50K+  →  Reagents Supplied
```

### Reveal on Scroll — General Rule
Every element that isn't immediately visible on load uses `data-aos` attribute or GSAP ScrollTrigger. Nothing appears static before it's scrolled to.

## PERFORMANCE & TECHNICAL REQUIREMENTS

- All images: `loading="lazy"`, `decoding="async"`, WebP format preferred
- Fonts: `<link rel="preconnect">` and `font-display: swap`
- Three.js canvas: `requestAnimationFrame` with `renderer.setPixelRatio(Math.min(window.devicePixelRatio, 2))` to avoid over-rendering on HiDPI screens
- `@media (prefers-reduced-motion: reduce)` — disable all GSAP animations, keep layout
- Responsive breakpoints: 375px, 768px, 1024px, 1280px, 1440px, 1920px
- No horizontal scroll on any breakpoint
- Tab key navigation works on all interactive elements
- All images have descriptive `alt` attributes
- ARIA labels on all icon-only buttons
